# Présentation de Battle Quiz Battle

Jeu de cartes qui comportent chacune des statistiques (points de vie, attaque, effets bonus…) en 1 vs 1, 
chaque joueur joue chacun son tour une carte par tour où le but est d’obtenir 3 points pour gagner. 

Une carte vaincue par l’adversaire offre un point au joueur, 
pour attaquer, le joueur doit utiliser sa carte en répondant préalablement à une question, 
s’il échoue alors l’attaque n’a pas lieu. 

Le joueur aura le choix entre une question facile, moyenne, ou difficile de culture générale, 
plus la difficulté choisie sera grande, plus forte sera l’attaque. 

# Utilisation de Battle Quiz Deluxe

Afin d'utiliser le projet, il suffit de taper les commandes suivantes dans un terminal :

ijavac main.java
ijava main

Le jeux est à lancer avec le terminal en plein écran.
En cas contraire il est possible d'avoir des problèmes d'affichage (si l'écran est trop petit).


```
./compile.sh
```
Permet la compilation des fichiers présents dans 'src' et création des fichiers '.class' dans 'classes'

```
./run.sh
```
Permet le lancement du jeu

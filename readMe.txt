# Présentation de Quiz Battle

Le principe du jeu est un combat de carte sur plateau au tour par tour, le but étant d'obtenir 3 points pour gagner.
Pour gagner un points il faut battre une carte monstre adversaire qui détient une certaine quantité de points de vie.
Pour attaquer une carte monstre adversaire, le joueur devra répondre à un QCM d'une difficulté variable en fonction de 2 points :
La première sera le niveau de la classe que le joueur aura renseigné au préalable( CE1 - CE2 - CM1 - CM2).
La seconde porte sur le choix du joueur entre facile, normal, et difficile.
Ce système permet une équité dans la difficulté des questions et permet très bien à des élèves de niveau de classe différente de s'affronter.
D'autres mécaniques propres au jeu sont à découvrir en jouant.

Des captures d'écran illustrant le fonctionnement du logiciel sont proposées dans le répertoire shots.


# Utilisation de Quiz Battle

Afin d'utiliser le projet, il suffit de taper les commandes suivantes dans un terminal :

```
./compile.sh
```
Permet la compilation des fichiers présents dans 'src' et création des fichiers '.class' dans 'classes'

```
./run.sh
```
Permet le lancement du jeu
